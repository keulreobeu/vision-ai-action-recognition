# 1_camera

Raspberry Pi 환경에서 카메라 영상을 수집하고, 작업 이벤트를 함께 기록하기 위한 폴더입니다.

---

## 1. 역할

이 폴더는 프로젝트의 데이터 수집 단계를 담당합니다.  
Raspberry Pi에 연결된 카메라를 이용해 작업 영상을 촬영하고, 동시에 이벤트 플래그를 기록하여 이후 전처리와 학습에 사용할 원본 데이터를 생성합니다.

주요 기능은 다음과 같습니다.

- Raspberry Pi 카메라 또는 연결된 카메라 장치로 영상 수집
- 프레임 저장 또는 영상 저장
- 작업 중 발생한 이벤트를 키 입력으로 기록
- 시나리오별 데이터셋 생성

---

## 2. 실행 환경

이 코드는 **Windows가 아닌 Raspberry Pi 환경에서 실행한 코드**입니다.

예상 실행 환경:
- Raspberry Pi
- Raspberry Pi OS
- Python 3.12
- OpenCV
- 연결된 카메라 장치
- 키 입력 또는 간단한 인터페이스를 통한 이벤트 기록
