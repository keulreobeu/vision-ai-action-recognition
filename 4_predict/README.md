# 4_predict

학습된 모델을 기반으로 예측을 수행하고, 결과를 후처리 및 평가하는 폴더입니다.

---

## 1. 역할

이 폴더는 모델 추론과 결과 해석 단계를 담당합니다.

주요 기능:
- 테스트 데이터 예측
- 프레임 단위 결과 생성
- TCN / YOLO 결과 결합
- 이벤트 단위 후처리
- 성능 평가

---

## 2. 입력 데이터

- 학습된 모델 가중치
- 테스트용 프레임 또는 텐서 데이터
- GT 라벨 CSV
- 객체 탐지 결과 또는 보조 입력 데이터

---

## 3. 출력 데이터

- 프레임 단위 예측 CSV
- 이벤트 단위 결과 CSV
- 성능 평가 결과
- 후처리 결과 파일

---

## 4. 처리 흐름

### 1) 모델 불러오기
학습된 체크포인트를 로드합니다.

### 2) 프레임 또는 시계열 예측
입력 데이터에 대해 예측을 수행합니다.

### 3) 결과 병합
행동 인식 결과와 객체 탐지 결과를 결합합니다.

### 4) 이벤트 후처리
프레임 단위 결과를 의미 있는 이벤트 단위로 변환합니다.

### 5) 평가
GT와 비교하여 성능을 확인합니다.

---

## 5. README에 적으면 좋은 내용

- 예측 대상 파일 구조
- GT와 PRED의 저장 위치
- 결과 CSV 컬럼 설명
- 후처리 기준
- 이벤트 판단 규칙

---

## 6. 주의사항

- 결과 저장 경로가 절대경로로 되어 있다면 상대경로로 수정해야 합니다.
- 파일명 오타는 정리하는 것이 좋습니다.
- GT 형식과 PRED 형식이 다르면 평가 시 오류가 날 수 있으므로 컬럼 정의를 문서화해야 합니다.

- 예측 결과 포맷 통일
