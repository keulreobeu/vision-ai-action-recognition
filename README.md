# GNP 자동 문서화 프로젝트

영상 기반 작업 행동 인식과 객체 상태 분석을 통해 이벤트를 추론하고, 최종적으로 문서를 자동 생성하는 AI 기반 문서 자동화 프로젝트입니다.

본 프로젝트는 작업 영상을 입력으로 받아 다음 과정을 수행합니다.

1. 카메라 또는 저장된 영상으로부터 데이터를 수집합니다.
2. 프레임, 랜드마크, 라벨 등 학습용 데이터를 전처리합니다.
3. 행동 인식 모델(TCN 계열)과 객체 탐지 모델(YOLO 계열)을 학습합니다.
4. 예측 결과를 후처리하여 이벤트 단위 결과를 생성합니다.
5. 예측 결과를 바탕으로 기록 문서를 자동 생성합니다.

---

## 1. 프로젝트 목표

기존에는 사람이 직접 영상 내용을 확인하고 기록 문서를 작성해야 했던 작업을,
컴퓨터 비전과 시계열 모델, 객체 탐지, LLM 기반 문서 자동화를 연결하여
실질적인 업무 자동화 흐름으로 구현하는 것을 목표로 합니다.

목적은 다음과 같습니다.

- 영상 데이터로부터 작업 행동을 자동 인식
- 객체 상태 및 작업 이벤트를 자동 추론
- 결과를 정리하여 기록 문서 자동 생성
- 반복적인 문서 작성 업무 효율화

---

## 2. 프로젝트 구조

```text
sessac_project/
├─ 1_camera/            # 영상 수집 및 이벤트 플래그 기록
├─ 2_preprocessing/     # 프레임/랜드마크/라벨/텐서 전처리
├─ 3_models/            # 행동 인식 및 객체 탐지 모델 학습
├─ 4_predict/           # 예측, 후처리, 평가
├─ 5_langchain/         # 문서 자동 생성
├─ video/               # 원본 영상 및 프레임 데이터
├─ data/                # CSV, NPZ 등 중간 산출물
├─ docker/              # Docker 환경 구성 파일
└─ output/              # 최종 생성 문서 및 결과물
```
