# 5_langchain

예측 결과를 바탕으로 기록 문서와 로그를 자동 생성하는 폴더입니다.

---

## 1. 역할

이 폴더에서는 모델 예측 결과를 사람이 바로 활용할 수 있는 문서 형태로 변환합니다.

주요 기능:
- 예측 CSV 해석
- 이벤트 로그 정리
- 이상 이벤트 생성
- 템플릿 문서 자동 작성
- 결과 파일 저장

---

## 2. 입력 데이터

- 예측 결과 CSV
- 이벤트 로그 CSV
- 템플릿 문서(DOCX, XLSX 등)
- API Key 또는 환경변수

---

## 3. 출력 데이터

- 자동 생성 문서
- 결과 요약 CSV
- 이상 이벤트 기록 파일
- 최종 보고용 산출물

---

## 4. 처리 흐름

### 1) 예측 결과 로드
이전 단계에서 생성된 예측 CSV를 읽습니다.

### 2) 이벤트 정리
예측 결과를 문서에 넣기 적절한 구조로 정리합니다.

### 3) 템플릿 매핑
문서 템플릿의 항목과 결과값을 연결합니다.

### 4) 문서 생성
DOCX, XLSX, TXT 등의 결과 문서를 생성합니다.
